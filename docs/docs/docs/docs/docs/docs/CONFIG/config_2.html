配置2：Linux环境

Linux环境对于生物信息学而言，是一个必备的环境，所以我们得确保自己的电脑有一个可以用的linux环境。在这里，我推荐wsl，这是微软官方发布的一个专门用于linux分析用的环境，可以说，这避免了大量的vmware的bug出现，对于小白也是相当友好的。

为了简单起见，我这里放上微软的官方教程：https://docs.microsoft.com/zh-cn/windows/wsl/install
